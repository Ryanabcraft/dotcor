Instalação

Instale com um comando.

Cole no PowerShell ou CMD. O instalador cria backup do config.toml, evita duplicar se o plugin já estiver ativo, adiciona o marketplace quando precisar e deixa tudo habilitado. 

powershell -c "irm https://ryanabcraft.github.io/doctor/install.ps1|iex"

Já veio instalado?

Sem entrada duplicada.

Se o Project Doctor já estiver instalado e ativado por outro marketplace, o instalador reconhece isso, mostra o nome encontrado e não cria outra entrada no config.toml.

Manual

Ou adicione no config.

Cole este bloco no arquivo C:\Users\SEU_USUARIO\.codex\config.toml, reinicie o Codex e procure por Project Doctor na tela de plugins. 

[marketplaces.ryan-plugins]
source_type = "git"
source = "https://github.com/Ryanabcraft/project-doctor-marketplace.git"
